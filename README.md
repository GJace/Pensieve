Pensieve
a flow of consciousness

THIS IS A WORK IN PROGRESS

I wanted to have a place where I could stroe my thoughts. The first name that came to mind was of course "thoughts". After a while my Pottery Magical memories kicked in, and I settled for Pensieve. In the Potterverse, Pensieves "were a magical device used to store and review memories. Pensieves were rare because only the most advanced wizards ever use them, and because the majority of wizardkind is afraid of doing so." (from Harry Potter wiki)

I wanted something that was extremely not complicated, front-end wise, html and css. A few reasons. I had not worked with just html/css since Uni times. I wanted to free myself of web frameworks for once. Mainly for pride reasons, but also because of resources/speed reasons.

Another reason to get back to "origins" is because llms are very good at working with html/css. We started ditching those and migrating to frameworks for a variety of reasons:
- simpler and more readable code
- reduce the amount of repetitive code
Now that llms can write code, there is not a need for a framework - for this kind of project.
